# このプロジェクトについて
2024年より日本大富豪連盟で導入された、大富豪の3人ルールを遊ぶためのコードが記載されています。

また、PCで実行することを前提にしております。

## 対象
**大富豪の3人で遊ぶ場合の公式ルールを知っている、または知る意思がある人**が対象です。※プロジェクト内でルールの表記は行っていないため

## 注意
必要最低限の機能のみ実装しています。以下の内容以外は実装されていません。
- 3人で大富豪を遊ぶための公式ルール（カードシャッフル、ローカルルールなど）
- プレイヤー1人、CPU2人での対戦
- 1～4回戦の得点の保持

最終的な手直しはしていますが、ほぼCopilotを用いてコードを記載しています。可読性や効率が悪いコードになっている可能性があることをご了承ください。

このコードを用いて3人大富豪のアプリやブラウザゲームを作っても大丈夫です。リリースされたら遊びたいので、報告お願いします。（プレイヤー同士で対戦できるようになっていると嬉しいです）

コードは自由に使って構いませんが、**個人利用のみ可能な**画像が使われています。このコードを用いてゲームを作成する場合は、imagesフォルダにある画像の削除、差し替えをよろしくお願いいたします。

## 起動方法（エンジニア向け）
このプロジェクトは**Node.js**および**React**で作成されています。Node.jsがインストールされていれば、ローカルで動かすことができます。

1. プロジェクトをダウンロードしてください。
2. プロジェクトディレクトリに移動し、以下のコマンドを実行してください:
   ```
   npm install
   ```
3. 依存関係がインストールされたら、以下のコマンドでプロジェクトを起動してください:
   ```
   npm start
   ```
ロジックで不備等がありましたら連絡お願いします。

# オマケ～大富豪について～

## 基本ルール

大富豪は、カードを使った競技的なゲームです。ゲームの目的は、自分の手札をできるだけ早く0枚にすることです。手札が0枚になると「大富豪」となり、次のゲームで有利な立場になります。逆に、手札を最後まで残してしまうと「大貧民」となり、次のゲームで不利になります。

### 1. ゲームの準備

大富豪は通常、3人以上でプレイされます。ゲームでは、**1組のカード（52枚）**を使い、各プレイヤーに均等にカードが配られます。カードは、数字によって強さが決まります。  
数字の強さは、3が一番弱く、2が一番強いです。順番は3～10＜J～K＜A＜2となります。  
Jokerは2より強い数字として扱ったり、他の数字として代用したりすることができます。また、スート（♤、♡、♢、♧）による強さの違いはありません。

### 2. ゲームの進行

#### ゲームの流れ

1. **最初のプレイヤー**がカードを1枚出します。  
同じ数字があったり、同じスートで連続した数字が3枚以上ある（階段）場合は、複数枚出すことができます。

2. 次のプレイヤーは、**出されたカードと同じ枚数**で、より強い数字のカードを出さなければなりません。  
例えば、出されたカードが同じ数字の2枚組であれば、同じ数字の2枚組を出す必要があります。また、階段の場合は、同じ枚数の階段を出さなければなりません。

3. 出せない場合、そのプレイヤーはカードを出すことができますが、出したくない場合も**パスを行うことができます**。

4. すべてのプレイヤーがパスをするか、カードを出しきった場合、**最後にカードを出したプレイヤー以外がパスをした場合**、最後にカードを出したプレイヤーが「親」になり、新たにカードを出します。

5. **最初に手札をすべて出し切ったプレイヤー**が「大富豪」となり、次のラウンドで最も有利な立場になります。次に「富豪」、次は「貧民」、最後に「大貧民」の順位がつきます。

### 3. カード交換

ゲーム終了後、カードを配り、次のゲームを開始する前に、大富豪は大貧民に2枚好きなカードを渡します。大貧民は、大富豪に2枚手札の中で一番強いカードを渡します。同様に、富豪は貧民に1枚好きなカードを、貧民は富豪に1枚手札の中で一番強いカードを渡します。  
カード交換が完了したら、大貧民が「親」としてゲームを開始します。

### 4. 勝利条件

手札を一番早く0枚にしたプレイヤーが勝利します。このプレイヤーは「大富豪」となり、次のゲームで他のプレイヤーよりも有利な立場になります。  
勝利した後も、引き続き次のゲームで他のプレイヤーと競い合い、最終的に階級に応じて手に入れた得点が最も高いプレイヤーがゲーム全体の勝者となります。

## 公式ルール

- **革命(同数)**  
同じ数字のカードを4枚出すと、カードの強さが逆転します。  
カードの強さは即時反映され、例えば3＜2の状態で9を使って革命が発生した場合、8以下のカードを4枚出せば革命返しが可能です。  
Jokerを数字の代わりに使って革命を起こしたり、Jokerを追加して5枚以上で革命を行うこともできます。

- **8切り**  
8を出すと場が流れ、その後8を出したプレイヤーが親になります。

- **都落ち**  
大富豪だったプレイヤーが次の試合で大富豪になれなかった場合、強制的に最下位（大貧民）となります。

- **縛り（スート縛り）**  
同じスートのカードを出すと、場が流れるまで同じスートのカードしか出せなくなります。  
ペアなどで2種類以上のスートがある場合、両方とも同じスートであれば縛りが発生します。  
縛り中にJokerを使うことはできますが、Jokerを使って縛ることはできません。

- **スペ3返し**  
Joker単体が出ている場合、♤3を出すことで場が流れ、その後♤3を出したプレイヤーが親になります。  
Jokerが出る前の縛りは適用されません。

また、ローカルルールとして明記されていませんが、注意すべきルールがいくつかあります。

- **階段**  
同じスートで連続する数字が3枚以上ある場合、それをまとめて出すことができます。  
例えば♢の9、10、Jがある場合、それをまとめて出すことができます。次のプレイヤーはこの階段より強い数字の階段を出さなければなりません。  
2と3をつなぐことはできません。また、8切りや革命といった効果は発生しませんが、スート縛りは適用されます。

- **禁止上がり**  
最強（通常は2、革命時は3）のカード、Joker、8切り、♤3単品で上がると禁止上がりとなり、次の試合で大貧民になります。  
都落ちが発生した場合でも、禁止上がりの方が優先されます（都落ちしたプレイヤーは貧民となります）。  
8を含む階段や♤3を含む階段・複数枚出しの場合、禁止上がりにはなりません。

- **Joker、ブラインドカード**  
Joker以外のカードをランダムで2枚抜き、  Jokerを2枚追加して、プレイヤーの手札が13枚になるよう調整します。

- **パスの制限**  
  一度パスをすると、場が流れて親が新しいカードを出すまでパスはできません。  
  このルールは、大富豪における駆け引きを増やす重要な要素だと思います。

## 3人ルールの場合に追加・変更されるルール
- **スート**  
   ♢、♡、♤の3種類のスートのみ使用されます。

- **Joker、ブラインドカード**  
   Jokerおよびブラインドカードはそれぞれ1枚となり、プレイヤーの手札は13枚に揃えられます。

- **革命**  
   同じ数字のカード3枚で革命が発生します。Jokerを数字の代わりに使ったり、Jokerを追加して4枚革命にすることも可能です。

- **交換枚数**  
   大富豪（1位）と大貧民（3位）の交換枚数は1枚です。富豪（2位）はカード交換を行いません。
